# APP Base

AppBase is a API base for building node apis.
This base use Node, Express, Mysql with Sequelize.

## Functionalities

- CRUD USER
- Login with GWT

## Getting Started

- Install node
- Install yarn

Clone or download this repository and run comands:

```bash
yarn
```

Configure ESLint:
